# Квалификация 4S06130103: «Разработчик программного обеспечения»

## Основная информация
* **Форма обучения:** Очная, Дистанционная
* **Язык обучения:** Русский, Казахский

---

## Описание программы

Программа направлена на подготовку востребованных специалистов по созданию современных сайтов, онлайн-сервисов и приложений. Студенты получают комплексную подготовку, охватывающую как классическую веб-разработку, так и передовые IT-направления.

---

## Изучаемые технологии и направления

### 1. Веб-разработка и Backend
* **Языки программирования:** HTML, CSS, JavaScript, Python, Java, PHP
* **Фреймворки:** React, Angular, Vue
* **Серверные технологии:** Node.js, Django, Spring
* **Базы данных:** MySQL, PostgreSQL
* **Инструменты и технологии:** Git, REST API, Docker

### 2. Разработка ПО и мобильных приложений
* **Языки программирования:** C#, Python, Java, Kotlin, Go
* **Платформы:** 1С
* **Базы данных:** MongoDB, MySQL

### 3. Компьютерная графика и дизайн
* **Инструменты:** Blender, CorelDRAW, Photoshop

### 4. Разработка игр и виртуальная реальность (VR)
* **Разработка:** Unity, создание интерактивных приложений
* **Оборудование:** VR-устройства (включая Oculus)

### 5. Передовые технологии и безопасность
* **Технологии:** Искусственный интеллект (AI), Big Data, Data Science
* **Безопасность:** Компьютерные сети и основы защиты информационной безопасности
